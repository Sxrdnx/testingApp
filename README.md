 ## TestingApp
App for improving my testing android skills where, I make tests like unit tests, integration tests, ui tests.
The tests that I was able to do are:
* Database tests
* Viewmodel test 
* Navigation tests
* Repository test
* Fragment test 

The main components and libraries that I use are: 
* Espresso
* Mockito
* Hilt
* Coroutines
* Room
* Retrofit
